# App Store Optimization (Play Store listing)

Paste these into **Play Console → Main store listing**. ASO ranking weight roughly:
**app title > short description > long description > reviews/installs**. Keywords in the
**title** count most, so the title carries the primary term ("Live Cricket Line").

> Honest note: you can't reliably rank for a rival's brand name ("cricline") — Play search
> rewards the *category* terms users actually type. People searching "cricline" want a
> **live-line** cricket app, so owning **"live cricket line" / "ball by ball" / "live score"**
> captures that intent legitimately. Don't put a competitor's name in your listing (Play can
> reject it).

---

## Title (max 30 chars)
```
LiveLine Guru: Cricket Score
```
(28/30 — the brand already carries the "live line" intent, so the keyword slot adds "cricket
score", the highest-volume term.)

## Short description (max 80 chars)
```
Fast live cricket line: ball-by-ball scores, session, scorecard. Free, no login.
```

## Long description (max 4000 chars)
```
LiveLine Guru is a fast, free live cricket line — real-time scores and ball-by-ball updates for
every match, with no login required.

Follow the live line as it happens: live cricket score, ball-by-ball commentary, current run
rate and required run rate, target and session/rate analytics, full scorecard, fall of wickets,
playing XI / squads, fixtures, results, series and points tables.

WHY CRICKETFAST
• Live line, fast — scores and ball-by-ball stream in real time, over by over
• Ball-by-ball — every delivery with a clean live timeline

• Full scorecard — batting, bowling, extras, fall of wickets
• Squads & playing XI for every match
• Fixtures, results, series and points tables
• Save matches with a tap — optional alerts when a wicket falls
• Free, lightweight, and works without an account

FORMATS & LEAGUES
Test, ODI and T20 — internationals, the IPL and major franchise leagues.

Looking for a fast live-line cricket app? LiveLine Guru keeps the live score, ball-by-ball and
session stats one tap away — free, no sign-up, no clutter.

Live cricket scores. Live line. Ball by ball. Session & rates. Scorecard. Free.
```

Primary: `live cricket line`, `cricket live line`, `live line`, `ball by ball`,
`live cricket score`, `live score`.
Secondary: `cricket session`, `cricket rates`, `scorecard`, `squads`, `points table`,
`IPL live score`, `T20 live`, `fastest live score`, `cricket live line app`.

## Other ASO levers (outside this repo)
- **Reviews & installs** drive ranking most after text — ask early users to rate.
- **Localized listings** (Hindi + regional) widen reach in the India market.
- **Screenshots + feature graphic** with the words "Live Line / Ball by Ball" on them.
- Keep the **`app.json` description** (used for some metadata) aligned with the short
  description above — already updated.
